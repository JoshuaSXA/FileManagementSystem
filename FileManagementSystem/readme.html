User Manual

1. Format System

In the simulator page, there is a button entitled as 'Format' in the touch bar. If you click this button, then the file-management-system will be formatted. This also indicates that all your file data will be lost.

2. Create Folder

In the simulator page, there is a button entitled as 'New Folder' in the touch bar. If you click this button, then a new window will be displayed. This window mainly contains a input label where you can input filename. If the filename you input is rational and correct, (This means the filename shouldn't be null or contain any special character.) a new folder will be created and displayed in the page after you click the 'Confirm' button.

3. Create File

In the simulator page, there is a button entitled as 'New File' in the touch bar. If you click this button, then a new window will be displayed. The subsequent steps are the same as that in part two. The only difference is that you will get a text file if the file is created successfully.

4. Delete File

It's very easy if you wanna remove a folder or file from your file system. There are only two steps you should follow: 1. Move your mouse right over the folder or file icon, and right-click the icon. 2. Select the option 'Delete', and click it. As you can see, the target file will be removed from your file system only if there isn't any system error.

5. Display File Contents

There are two ways to open and display the folder or text file contents. First, you can double-click the target file, then the file will be opened. Second, you can also get the file content showed by right-clicking the file icon. Because of different file type, the content of folder and text file is totally different.

6. Change File Directory

You should use the "Cut" and "Paste" function to realize file directory modification. First, you are supposed to cut the target file. The system will save the file into the clipboard. Second, int the target file directory, you should paste the file. Not surprisingly, the target file directory structure has been changed.

7. Return to Previous Directory

You can click the return button to come back to last file directory.

8. Write File

To begin with, you should open a text file and click the blank editable area. In edit mode, you can input any text as you want, but the number of characters in your text should be limited no more than 1024.

9. Save File

In edit mode, there is a button entitled as "Save" in the touch bar. Every time you finish yout text editing, you are advised to click this button before exiting or your text data would be lost.

9. Search File

If your file directory structure is complex and you wanna search a file with specific filename. We provide you with the file-search function. You can input the filename in the search line, and click the search icon on the right. As you can see, all the matching results will be displayed.
